# Моделирование трёхмерного процесса теплопроводности

Курсовой проект по дисциплине **«Параллельное программирование на суперкомпьютерных системах»**.

## О проекте

Проект реализует численное решение трёхмерного нестационарного уравнения теплопроводности (уравнения диффузии) для однородной среды. Используется явная конечно-разностная схема (FTCS). В центре расчетной области инициализируется горячий куб, и моделируется процесс распространения тепла.

## Реализации

Проект включает несколько параллельных реализаций для сравнения производительности:

- **C & MPI** (распределенная память, декомпозиция по оси Z)
- **C & OpenMP** (разделяемая память, автоматическое распараллеливание циклов)  
- **C & Pthreads** (разделяемая память, ручное управление потоками с декомпозицией по осям X и Y)
- **Python & MPI** (реализация на Python с использованием `mpi4py` и `numpy`)

## Структура репозитория

- `common.h` – общие параметры и константы для всех реализаций
- `heat3d_mpi.c` – реализация на C с использованием MPI
- `heat3d_openmp.c` – реализация на C с использованием OpenMP
- `heat3d_pthreads.c` – реализация на C с использованием Pthreads
- `heat3d_mpi.py` – реализация на Python с использованием mpi4py
- `media/` – директория с графиками и визуализациями
